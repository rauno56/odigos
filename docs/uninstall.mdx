<Note>
  Odigos uninstallation process is still work in progress. We are working on
  making this process easier by transitioning from a helm chart installation to
  CLI based installation.
</Note>

### Uninstalling Odigos

Install the [Odigos CLI](https://github.com/odigos-io/odigos/tree/main/cli) and execute the following command:

```bash
odigos uninstall
```
